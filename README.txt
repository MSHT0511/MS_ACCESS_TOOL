# MS_ACCESS_TOOL
大嫌いなACCESSの開発でちょっとした助けになるモジュール等を管理

ACCESS初心者で死ぬほどACCESS嫌いだけど、いやいや開発しているあなたの助けになればと思っております。

▼使い方
VBEからインポートしてください。
開発用の補助的なモジュールのため、マクロに「ExportVbaProgramCode」を登録して実行。
または、VBE※上で「ExportVbaProgramCode」にカーソルをフォーカスしてF5キーを押下するとプログラムが実行されます。（知ってるか(;^ω^)）
実行したACCESSのフォルダに「_VBA_[実行したACCESSのファイル名]」というフォルダが作成されるため、そのフォルダにプログラムファイルが保存されます。

▼注意
・保存先に同名のプログラムファイルがある場合は上書きされます。


【用語】
※VBE・・・・Visual Vasic Editor
  使いづらいをぎゅっとしたVBA開発用のエディタのことです。
